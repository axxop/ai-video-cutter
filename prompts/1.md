# 短视频文案创作 Prompt（精简版）

## 一、文案结构

1. **开场钩子（前3-5句）**  
   - 制造悬念：反常现象 / 极端对比  
   - 快速定位：点明主题 + 最大看点  
   - 情绪词：巅峰、最强、封神、逆天、离谱等  
   - 可搭配：
     - 数字：40分钟、4个半人、一口气X个
     - 悬念：万万没想到、细思极恐、真相竟是
     - 禁忌：千万别、慎点、看完睡不着

   开场公式示例：
   - 「数字+反常」：40分钟速杀4个半人？这集直接疯了！
   - 「疑问+悬念」：凶手为什么要杀同伙？真相细思极恐…
   - 「禁止+诱惑」：千万别在深夜看这集！看完整晚失眠。

2. **核心卖点（紧随开场）**  
   - 用数字强调：时长、人数、次数等  
   - 用句式放大冲突：
     - 很难想象在短短40分钟里…
     - 出乎意料的是…
   - 直接回答：为什么这集值得看？

3. **故事主线（占60-70%）**  
   - 线性叙事：按时间推进  
   - 常用衔接词：随后、紧接着、与此同时、没过多久、就在这时  
   - 每段有小高潮：悬念递进、信息不断更新  
   - 允许多线并行：A线案件进展，B线人物心理 / 其他角色。

4. **高潮揭秘（15-20%）**  
   - 用「原来」「事实上」引出真相  
   - 先讲结论，再补逻辑推导和细节  
   - 把前面埋的伏笔、疑点一一解释清楚。

5. **结尾收束（最后2-3句）**  
   - 情感升华：回到人性、选择、代价等  
   - 补一两个小彩蛋或后续走向  
   - 呼应开头，形成闭环。

## 二、输入输出格式

**输入（字幕行格式）：**
```
1 第一句话内容
2 第二句话内容
3 第三句话内容
...
```

---

### ⚠️ **重要：输出格式规范**

**必须严格按照以下格式输出：**

```
[10s] [1-3] 第一到第三句文案内容
[20s] [4-7] 第四到第七句文案内容
[30s] [8-12] 第八到第十二句文案内容
```

**格式说明：**
- **[时间标记]**：表示这段内容的**说话时长**（非累计时间），如 `[15s]` 表示这段话需要说 15 秒
  - 按照正常语速约 **5 字/秒** 计算，75 字内容约需 15 秒
- **[行号范围]**：原始字幕的单个连续行号范围，如 `[1-50]` `[51-120]`，表示这段内容对应的原始字幕范围
  - **使用单个连续范围**，格式为 `[起始行-结束行]`，不要用逗号分隔多个范围
- **段落内容**：将选取的句子重组为流畅的口播文案，保持信息完整

**格式要求：**
1. ✅ 每段必须包含：`[时间] [行号] 内容` 三部分
2. ✅ **时间标记表示该段的说话时长**，按 **5 字/秒** 计算（50 字 ≈ 10 秒，75 字 ≈ 15 秒）
3. ✅ **每行内容控制在 60-90 字**，精炼扼要，每段说 12-18 秒
4. ✅ **行号范围必须是单个连续范围**，格式 `[1-50]`，不要用 `[1-3,5-8]` 这种多段格式
5. ✅ **尽量多分段**，至少 20-30 段，每段覆盖一个小情节或知识点
6. ✅ **段落之间不要有空行**，每段紧密相连，一行一段
7. ✅ 内容要连贯顺口，适合口播朗读
8. ✅ 保留核心信息，允许轻微润色和衔接优化
9. ❌ 不要省略时间标记或行号标记
10. ❌ 不要改变格式结构
11. ❌ 单行内容不要超过 110 字（22 秒）

**示例对比：**

❌ **错误格式**（缺少标记、有空行、行号用逗号分隔）：
```
第一段内容...

[10s] [1-3,5-7] 第二段内容，行号格式错误...
```

✅ **正确格式**（按 5 字/秒计算，单个连续行号范围，无空行）：
```
[15s] [1-50] 第一段内容，约 75 字，从第 1 行到第 50 行的内容重组而成，需要 15 秒说完。开场制造悬念，快速点明主题...
[18s] [51-120] 第二段内容，约 90 字，从第 51 行到第 120 行的内容整合，需要 18 秒说完。故事主线推进，设置关键冲突...
[14s] [121-180] 第三段内容，约 70 字，从第 121 行到第 180 行的内容提炼，需要 14 秒说完。揭示核心真相，解释疑点...
```

## 三、创作指令模板

```
请创作一篇【主题】的短视频解说文案，要求：

1）开场设计（3-5句，15-20秒）
- 用反常现象 / 强对比开场，制造悬念
- 快速点明本期主题和最大看点
- 使用极端词（巅峰、最强、唯一、从未等）拉高期待

2）核心卖点（2-3句，约10秒）
- 用具体数字或量化描述突出不寻常之处
- 用“很难想象…”，“出乎意料的是…”强化冲突
- 点明：为什么这一集 / 这个故事值得看

3）故事主线（占60-70%）
- 采用时间线清晰的线性叙事
- 每30秒设置一个小高潮或反转
- 大量使用衔接词：随后、紧接着、与此同时、没过多久、就在这时
- 根据需要在不同角色 / 场景间切换视角
- 适当加入人物对话、心理活动提升代入感

4）高潮揭秘（15-20%）
- 用“原来…”，“事实上…”等句式抛出真相
- 交代清楚作案手法 / 行为逻辑 / 动机
- 前后呼应，解释之前埋下的所有关键疑点

5）结尾收束（2-3句，约10秒）
- 做一次情感升华或观点总结
- 给一个小彩蛋或后续想象空间
- 让结局在情绪上有一个“收住”的感觉

6）语言风格
- 以短句为主，长短句结合控节奏
- 口语化，避免生硬书面语
- 适当使用“竟然 / 居然 / 顿时 / 万万没想到”等情绪词
- 多用“只见 / 看着 / 听到这话 / 意识到”等，增强画面感

7）篇幅控制
- 总字数控制在 800-1200 字
- 视频时长约 3-5 分钟
- 开场 10-15%，主线 60-70%，高潮 15-20%，结尾 5-8%
```

## 四、常用技巧速查

- 悬念公式：
  - 对比：本来应该X，但结果却Y
  - 数字：短短X时间里，竟然发生了Y
  - 身份：看似是A，其实是B
  - 动机：为什么要这么做？

- 节奏控制：
  - 快节奏：5-8字短句连用
  - 慢节奏：15-20字长句描述关键画面
  - 转折：用“可 / 然而 / 结果 / 不料”瞬间拐弯
  - 同步推进：用“与此同时”做多线并行

- 高频情绪与评价词（适量使用）：
  - 悬念：竟然、居然、不料、突然、意外、万万没想到
  - 爆点：炸了、裂开、绷不住、给跪了、破防、离谱
  - 评价：绝了、无敌、天花板、封神、降维打击

- 常用句头：
  - 原来… / 就在这时… / 随后… / 与此同时…
  - 看着… / 听到这话… / 他这才意识到…
  - 更离谱的是… / 谁能想到… / 结果你猜怎么着？

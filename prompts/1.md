# 短视频文案创作Prompt

## 文案结构分析

基于提供的柯南剧集解说文案，该短视频文案具有以下核心结构特征：

### 1. 开场钩子（前3-5句）- 标题党核心技巧
- **制造悬念**：用反常现象或极端对比吸引注意力
- **快速定位**：明确告知视频主题和核心看点
- **情绪铺垫**：用"脱颖而出"、"巅峰"、"震惊"、"绝了"等词汇建立期待
- **标题党关键词**：
  * 程度词：史上最强、顶级、天花板、封神、绝绝子、逆天、离谱、离大谱
  * 悬念词：万万没想到、你敢相信、打死不说、谁懂啊、细思极恐、背后真相
  * 反转词：真相竟是、没想到、居然、实锤了、反转来了、惊天反转
  * 情绪词：炸裂、震撼、窒息、泪目、破防了、麻了、绷不住了、给跪了
  * 数字爆点：仅用X秒、短短X分钟、一口气X个、竟然X次
  * 禁忌词：千万别、别再、劝你别、千万不要、看完腿软、慎点
  * 对比词：秒杀、吊打、碾压、完爆、降维打击
- **劲爆开场公式**：
  * 「数字+反常」：40分钟速杀4个半人？这集柯南疯了！
  * 「疑问+悬念」：凶手为什么要杀自己的同伙？真相细思极恐...
  * 「禁止+诱惑」：千万别在深夜看这集！看完整晚失眠
  * 「反转预告」：前方高能预警！这反转我跪了
  * 「情绪宣泄」：绷不住了！这剧情也太离谱了吧！

示例（普通版）：
```
有很多人说柯南后期除了主线以外，所有的内容质量都不及前300集。
可就是在这样的口碑之下，却有一集后期原创剧集脱颖而出。
而她便是能面宅邸跳舞动，在播出之际她不仅收获了大量好评，多数观众甚至还将他评为了原创巅峰。
```

劲爆版改写：
```
都说柯南后期拉跨？这一集直接封神！
40分钟速杀4个半人，其中3个还是凶手？这剧情也太疯了吧！
更离谱的是，真正的凶手竟然用一把勺子毒杀了所有人，细思极恐...
```

### 2. 核心卖点（紧随开场）
- **突出亮点**：直接说明为什么值得看
- **数据化表达**：用具体数字增强说服力（如"40分钟"、"四个半人"、"三人是凶手"）
- **制造冲突**：强调不寻常、反直觉的元素

示例：
```
而他之所以能有如此好的口碑，不是因为案件手法有多么亮眼，而是因为里面有着最疯狂的剧情走向。
很难想象在短短40分钟的剧集里，凶手却能做到速杀四个半人，其中还有三人是真正的凶手。
```

### 3. 故事主线（占比最大）
- **线性叙事**：按时间顺序推进（"从...说起"、"随后"、"与此同时"）
- **节奏控制**：用短句加快节奏，长句描述关键场景
- **时间衔接词**：大量使用"就在这时"、"随后"、"与此同时"、"没过多久"等
- **悬念递进**：每个段落都有小高潮，层层推进
- **视角切换**：在多个角色/场景间快速切换制造紧张感

关键转折词：
- 引入新情节：原来、就在这时、与此同时
- 制造转折：可、然而、不料、好在
- 推进节奏：随后、紧接着、接着、之后

### 4. 解密揭秘（高潮部分）
- **真相揭示**：柯南推理过程
- **逻辑推导**：层层剖析手法和动机
- **细节回顾**：前后呼应，解释伏笔

示例：
```
这起案子乍看之下，犯人似乎都死干净，然而事实上，真正的犯人却还活得好好。
而这一点最好的证明就是第一起案件。
```

### 5. 结尾收束（最后2-3句）
- **情感升华**：回归人性或道理层面
- **彩蛋补充**：额外的有趣信息
- **完整闭环**：呼应开头或给出圆满结局

## 短视频文案创作Prompt模板

### 输入输出格式说明

**输入格式**（字幕格式）：
```
[1] 第一句话内容
[2] 第二句话内容
[3] 第三句话内容
...
```

**输出格式**（带时间轴的分段格式）：
```
[0-10秒] [行号1-3] 
第一句话内容第二句话内容第三句话内容

[10-20秒] [行号4-7]
第四句话内容第五句话内容第六句话内容第七句话内容

[20-30秒] [行号8-12]
第八句话内容...
```

**格式说明**：
- 每段标注时间范围（每段约10-15秒）和对应的原始行号范围
- 将连续的句子合并成流畅的段落
- 保持原文核心内容，但可优化表达和节奏
- 每段之间空一行

---

```
请创作一篇[主题]的短视频解说文案，要求：

【开场设计】（3-5句，15-20秒）
1. 用一个反常现象或强对比开场，制造悬念
2. 快速点明本期主题和最大看点
3. 用极端词汇（巅峰、最强、唯一、从未等）建立期待

【核心卖点】（2-3句，10秒）
1. 用具体数字或量化描述突出不寻常之处
2. 用"很难想象..."、"出乎意料的是..."等句式强化冲突感
3. 给出"为什么值得看"的核心理由

【故事主线】（占60-70%篇幅）
结构要求：
- 采用线性叙事，时间线清晰
- 每30秒设置一个小高潮或转折点
- 大量使用衔接词：
  * 时间推进：随后、紧接着、与此同时、没过多久
  * 转折制造：可、然而、不料、但、好在
  * 场景切换：另一边、与此同时、就在这时
- 保持快节奏，避免冗长描述
- 适时插入人物对话或心理活动

叙事技巧：
1. 多线叙事交织（A线、B线交替推进）
2. 悬念递进（一个悬念解决，立刻引入新悬念）
3. 视角切换（在不同角色间快速切换）
4. 细节铺垫（为后续反转埋下伏笔）

【高潮揭秘】（15-20%篇幅）
1. 用"原来"、"事实上"等词引入真相
2. 逻辑推导要清晰，手法解析要具体
3. 前后呼应，解释之前的疑点

【结尾收束】（2-3句，10秒）
1. 情感升华或给出启示
2. 补充有趣彩蛋或后续发展
3. 给观众一个满意的结局

【语言风格】
- 短句为主，长短句结合控制节奏
- 口语化表达，避免书面语
- 适当使用"竟然"、"居然"、"顿时"等情绪词
- 制造画面感，用"只见"、"看着"等引导想象

【篇幅控制】
- 总字数：800-1200字
- 对应视频时长：3-5分钟
- 开场占比：10-15%
- 主线占比：60-70%
- 高潮占比：15-20%
- 结尾占比：5-8%
```

## 关键技巧总结

### 悬念制造公式
1. **对比悬念**：A情况下应该X，但实际上却Y
2. **数字悬念**：在短短X时间/空间内，竟然发生了Y
3. **身份悬念**：看似是A，实际上是B
4. **动机悬念**：为什么要这么做？

### 节奏控制技巧
- **快节奏段落**：5-8字短句连用，制造紧张感
- **慢节奏段落**：15-20字长句，描述关键场景细节
- **转折加速**："可"、"然而"后立即引入冲突
- **同步推进**："与此同时"实现多线并进

### 情绪词库
- 悬念类：竟然、居然、不料、突然、意外、万万没想到、谁能想到
- 转折类：然而、可、但、好在、不料、没想到、结果、谁知道
- 强调类：顿时、瞬间、立刻、赶忙、迅速、直接、当场、秒速
- 程度类：十分、极其、非常、根本、压根、简直、完全、彻底
- 爆点类：炸了、裂开、麻了、绷不住、给跪了、破防、窒息、离谱
- 评价类：绝了、牛逼、无敌、天花板、封神、降维打击、吊打
- 引导类：注意看、重点来了、高能预警、前方核能、划重点

### 常用句式
1. "原来..." - 揭秘真相
2. "就在这时..." - 引入转折
3. "随后..." - 推进情节
4. "看着..." - 描述反应
5. "听到这话..." - 承接对话
6. "意识到..." - 表现觉察
7. "很难想象..." - 强化冲突
8. "与此同时..." - 平行叙事
9. "万万没想到..." - 爆点前置
10. "更离谱的是..." - 升级悬念
11. "谁能想到..." - 反转铺垫
12. "直接..." - 加速节奏（直接秒了、直接炸了）
13. "给你整个..." - 夸张描述
14. "这波..." - 评价操作（这波在大气层）
15. "细思极恐..." - 深度引导
16. "高能预警！" - 提示爆点
17. "结果你猜怎么着？" - 互动引导
18. "彻底..." - 强化结果（彻底麻了、彻底绷不住）
